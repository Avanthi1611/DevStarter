Welcome to Python Installation Guide

Last Updated: 15 April, 2025

Introduction

If you’re beginning your journey into Python programming, setting up Python on your machine is the essential first step. This guide will walk you through the installation process for Windows, macOS, and Linux. Whether you’re installing it for the first time or updating to the latest version, we’ve got you covered.

Before we dive in, let’s understand a few key things about Python:

  • Python is a popular, high-level, general-purpose programming language.
  • Each Python version is free and open-source under the GPL (General Public License).
  • You can download official releases from python.org.
  • Most Linux distributions include a package manager that simplifies installation.

1. Installing Python on Windows

  1. Visit the official Python website: Go to the downloads page.
  2. Select a version: Download the latest stable version (e.g., Python 3.13.2).
  3. Run the installer: Open the downloaded .exe file and ensure you check "Add Python to PATH".
  4. Complete installation: Click “Install Now.” Once done, open the command prompt and type python --version to confirm the installation.
  5. Optional - Launch IDLE: Open the Python Shell (IDLE) from the Start Menu and try printing something like Hello, Python!.

2. Installing Python on macOS

  1. Use Homebrew: Open Terminal and run: brew install python3.
  2. Verify installation: After installation, run python3 --version to check that Python was installed successfully.

3. Installing Python on Linux

Most Linux distributions come with Python pre-installed. To check if it’s available:

  • Check Python 2: python --version
  • Check Python 3: python3 --version

If you need to install or update Python, use the commands below (for Ubuntu-based systems):

sudo apt install software-properties-common -y
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt-get update
sudo apt-get install python3.13

To confirm installation:

python3.13 --version

4. Installing Python Modules (Beginner Friendly)

Python has a rich ecosystem of reusable code packages called “modules” or “libraries.” You can use pip (Python’s package installer) to easily install them.

What is pip?

pip is the default package manager for Python 3.4 and above.

Basic pip usage

  • pip install package – Installs the package
  • pip install "package>=1.0.0" – Installs a specific or newer version
  • pip install --upgrade package – Updates an installed package
  • pip install --user package – Installs for your user account only

Creating a Virtual Environment

To isolate dependencies per project:

python -m venv myenv

Then activate it:

  • Windows: myenv\Scripts\activate
  • macOS/Linux: source myenv/bin/activate

Having Multiple Python Versions?

If multiple versions exist on your machine, specify which one to use:

  • python3 -m pip install package
  • py -3.9 -m pip install package (on Windows)

pip Not Found?

If pip isn’t installed, use this command:

python -m ensurepip --default-pip

Advanced Packages

For packages like NumPy, Pandas, or Matplotlib, consider using Anaconda — a Python distribution designed for scientific computing.

5. Conclusion

Setting up Python on Windows, macOS, or Linux is a beginner-friendly process that enables you to begin writing and running Python code in no time. Whether you used an installer, Homebrew, or terminal commands, you now have a fully working Python environment.

Always remember to verify your installation using the version command in your terminal or command prompt. If something doesn’t work as expected, the official Python documentation is your best friend.

With Python successfully installed, you’re ready to dive into the world of coding and start building awesome projects. Good luck and happy coding!
