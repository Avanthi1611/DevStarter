Welcome to Node.js Installation Guide

Last Updated: 15 April, 2025

Installing Node.js on Windows is a straightforward process, but it’s essential to follow the right steps to ensure smooth setup and proper functioning of Node Package Manager (NPM), which is crucial for managing dependencies and packages. This guide will walk you through the official site, NVM, Windows Subsystem, and Package Manager for Windows 7, 8, 10, and 11.

Prerequisites for Installing Node.js on Windows

  • Windows OS: Windows 7 or later
  • Administrator Rights: Required for installation
  • Internet Connection: Stable connection for downloading
  • Windows Installer: Latest version for smooth installation
  • Text Editor/IDE: Visual Studio Code or similar
  • Basic Command Line Knowledge: CMD or PowerShell familiarity
  • Disk Space: At least 100 MB of free space
  • Hardware Requirements: Minimum 2 GB RAM and 1 GHz processor

Method 1: Use the Official Website

  1. Step 1: Download Node.js Installer

    Visit the official Node.js website to download the Node.js ‘.msi’ installer based on your system architecture (32-bit or 64-bit). The LTS version is recommended for stability.

  2. Step 2: Run the Installer

    Locate the downloaded file and follow the setup wizard. Ensure you install npm and native modules.

  3. Step 3: Finish Setup and Install Node.js and NPM

    Complete the installation and verify by running node -v and npm -v in Command Prompt or PowerShell.

Method 2: Install Node.js using NVM

  1. Step 1: Install NVM for Windows

    Download the NVM installer from GitHub and follow the instructions.

  2. Step 2: Install Node.js and NPM

    Use nvm install latest or specify a version, e.g., nvm install 14.17.6.

  3. Step 3: Verify Node Installation

    Run node -v to check the installed version.

Method 3: Use Windows Subsystem (WSL)

  1. Step 1: Open PowerShell

    Run PowerShell as Administrator and install WSL.

  2. Step 2: Set up a Linux Distribution

    Launch a Linux app like Ubuntu and set up your environment.

  3. Step 3: Install Node.js and NPM via apt

    Run sudo apt install nodejs npm to install Node.js and NPM.

Method 4: Install Node.js & NPM using WPM

  1. Step 1: Open PowerShell as Administrator

    Right-click on the Start Menu and select Windows PowerShell (Admin).

  2. Step 2: Run the Installation Command

    Use winget install OpenJS.NodeJS to install Node.js.

  3. Step 3: Verify Installation for Node.js and NPM

    Check the installation by running node -v and npm -v.

Conclusion

Installing Node.js and NPM on Windows is crucial for developers working in JavaScript and server-side applications. By managing your packages with NPM, you can streamline your development process and maintain your projects efficiently.
