Introduction

C is a widely-used programming language developed in the 1970s by Dennis Ritchie. It’s well-known for offering low-level access to memory, making it an ideal choice for systems programming, embedded systems, and operating system development.

To begin writing and running C code, you have two options:

  • Use an Online IDE for quick and hassle-free compilation
  • Set up a local development environment on your system

2. Install C Locally

For full control and offline development, setting up C on your machine involves two key components:

  • C Compiler: Translates your code into executable files
  • Code Editor (IDE): Where you write and organize your code

3. Installing GCC Compiler

On Linux

Use the following terminal commands to install GCC on Debian-based systems:

sudo apt update
sudo apt install build-essential

On Windows

Install GCC using MinGW:

  1. Download the MinGW installer
  2. Run the installer and select the GCC package
  3. Add MinGW\bin to your system’s PATH environment variable

On macOS

Use Homebrew to install GCC:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install gcc

4. Verify GCC Installation

Once installed, open your terminal or command prompt and run:

gcc --version

If everything was installed correctly, you’ll see the GCC version info displayed.

5. Installing a Code Editor (IDE)

While many IDEs are available, Visual Studio Code (VS Code) is lightweight, powerful, and perfect for C development.

Steps:

  1. Download the installer for your OS from the official site
  2. Install using default/recommended settings

Other IDE Options:

  • Atom Text Editor (Linux)
  • Xcode (macOS)
  • Code::Blocks (Windows/Linux)
  • Sublime Text (Windows)

6. Configure VS Code for C

Install Extensions

  1. Open the Extensions tab in VS Code
  2. Search and install: C/C++ Extension Pack and Code Runner

Enable Terminal Input

To allow user input during execution:

  1. Go to File → Preferences → Settings
  2. Click Extensions tab
  3. Search “Run in Terminal” and check the box

7. Run a “Hello World” Program

Let’s test everything with a simple example:

  1. Open a folder where you’ll save your C files
  2. Create a new file with a .c extension
  3. Paste the following code:
#include <stdio.h>

int main() {
    printf("Hello World");
    return 0;
}

Save the file and press Ctrl + Alt + N to run it using Code Runner.

If you see "Hello World" printed in the terminal, you're all set!

8. Conclusion

C is a powerful and foundational language that serves as the backbone for many modern technologies. Whether you use an online IDE for quick testing or set up a complete development environment on your system, you're now ready to begin writing and executing C programs.

As you continue learning, explore more about C syntax, functions, pointers, and memory management. Happy coding!
