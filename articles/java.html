Introduction

Java is a versatile programming language widely used for building applications. To start coding in Java, you first need to install the Java Development Kit (JDK) on your system. This article provides detailed steps for installing Java on Windows 7, 8, 10, 11, Linux Ubuntu, and macOS.

Download and Install Java – Windows Operating System

  1. Visit the official Oracle website: Go to Oracle's Java page and click on the "Download Java" icon.
  2. Download JDK 23: Select the Windows x64 Installer option to download the .exe file for 64-bit Windows OS.
  3. Run the installer: Double-click on the downloaded .exe file and follow the installation wizard. Ensure you set the environment variables as per the guide.
  4. Verify the installation: Open the command prompt and type java --version to confirm.

Download and Install Java – Linux

  1. Visit the Oracle website: Download the Linux version of the JDK as a .tar.gz file.
  2. Extract the file: Use the command tar -xvzf jdk-23_linux-x64_bin.tar.gz and move it to the /opt directory.
  3. Set environment variables: Edit the shell configuration file (e.g., .bashrc) to include the JAVA_HOME and update the PATH.
  4. Verify installation: Run java --version to check the installed version.

Download and Install Java – macOS

  1. Visit the Oracle website: Download the macOS version as a .dmg file and install it.
  2. Set environment variables: Edit the shell configuration file (e.g., .zshrc) to set JAVA_HOME and update the PATH.
  3. Verify installation: Open the terminal and type java --version to confirm.
